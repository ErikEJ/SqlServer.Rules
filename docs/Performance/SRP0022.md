## Description

## Summary
  
Consider using <c>RECOMPILE</c> query hint instead of <c>WITH RECOMPILE</c> option 
  
### Examples
  

               good:
                ```sql
                CREATE PROCEDURE dbo.my_proc
                BEGIN
                    SELECT col_A, col_b 
                    FROM some_complicated_set 
                    WHERE some_complicated_filter = 1 
                    OPTION(RECOMPILE)
                ```
            
               bad:
                ```sql
                CREATE PROCEDURE dbo.my_proc
                WITH RECOMPILE
                ```
