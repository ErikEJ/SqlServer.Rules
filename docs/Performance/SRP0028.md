## Description
  
Avoid explicit RANGE window frames; prefer explicit ROWS.
  
## Summary
  
Avoid explicit RANGE in window frames and prefer ROWS.
  
### Remarks
  
Explicit RANGE window frames can be slower than equivalent ROWS frames.
Prefer ROWS when row-based framing is intended.
