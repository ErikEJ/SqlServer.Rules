## Description
  
Cursors default to writable. Specify FAST_FORWARD.
  
## Summary
  
Cursors default to writable. Specify FAST_FORWARD.
  
### Examples
  
```sql
CREATE PROCEDURE dbo.CursorTest
AS
SET NOCOUNT ON;

DECLARE vendor_cursor CURSOR FOR 
SELECT Col1, Col2
FROM [dbo].[TestTableSSDT]
WHERE Col3=1
ORDER BY Col1;

-- SRP0030, SRP0016
```
  
### Remarks
  
Cursors are writable by default and can incur extra overhead.
For read-only, forward-only cursor scenarios, specify FAST_FORWARD.
