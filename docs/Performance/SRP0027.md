## Description
  
Avoid explicit conversion of columnar data in the WHERE clause. (Sargable)
  
## Summary
  
Avoid explicit conversion of columnar data in the WHERE clause.

SET NOCOUNT ON;

FROM [dbo].TestTableSSDT
