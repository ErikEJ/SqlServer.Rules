## Description
  
SET CONCAT_NULL_YIELDS_NULL should be ON.
  
## Summary
  
SET CONCAT_NULL_YIELDS_NULL should be ON.
  
### Examples
  
```sql
CREATE PROCEDURE [dbo].[ConcatNullYieldsNullOffTest]
AS
BEGIN
    SET NOCOUNT ON;
    SET CONCAT_NULL_YIELDS_NULL OFF;
END;

-- SRD0084
```
```sql
CREATE PROCEDURE dbo.Sets
AS
SET ANSI_NULLS  OFF
SET ANSI_PADDING OFF
SET ANSI_WARNINGS OFF
SET ARITHABORT OFF
SET NUMERIC_ROUNDABORT ON
SET QUOTED_IDENTIFIER OFF
SET CONCAT_NULL_YIELDS_NULL OFF

-- SRD0084, SRD0085, SRD0086, SRD0087, SRD0088, SRD0089
```
  
### Remarks
  
When CONCAT_NULL_YIELDS_NULL is OFF, concatenating a NULL value can produce unexpected
results and can prevent indexed views and indexes on computed columns.
