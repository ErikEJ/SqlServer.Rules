## Rule Information
  
|    |    |
|----|----|
| Id | SRD0700 |
| Friendly Name | Database PAGE_VERIFY option is not CHECKSUM |
| Category | Design |
| Ignorable | false |
| Applicable Types | Model  |
  
## Description
  
Set `<PageVerify>CHECKSUM</PageVerify>` in the project file to enable PAGE_VERIFY = CHECKSUM.
  
## Summary
  
Please review the PAGE_VERIFY database option
  
### Remarks
  
The database PAGE_VERIFY option should be set to CHECKSUM.
CHECKSUM detects more errors than TORN_PAGE_DETECTION or NONE, and is the recommended setting.
