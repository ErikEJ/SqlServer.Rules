## Rule Information
  
|    |    |
|----|----|
| Id | SRD0092 |
| Friendly Name | Avoid named primary key constraints on temp tables |
| Category | Design |
| Ignorable | true |
| Applicable Types | Procedure  |
|   | Scalar Function |
|   | Table Valued Function |
  
## Description
  
Primary Key Constraints on temporary tables should not be named.
  
## Summary
  
Primary key constraints on temporary tables should not be named.
  
### Examples
  
```sql
CREATE PROCEDURE [dbo].[NamedPKOnTempTable]
AS
BEGIN
    SET NOCOUNT ON;

    CREATE TABLE #Good
    (
        ID INT PRIMARY KEY
    );

    (
        ID INT,
        PRIMARY KEY (ID)
    );

    CREATE TABLE [dbo].[Foo#Bar]
    (
        ID INT,
        CONSTRAINT [PK_Foo#Bar] PRIMARY KEY (ID)
    );

    CREATE TABLE #Bad
    (
        ID INT,
        CONSTRAINT [PKID] PRIMARY KEY (ID)
    );
END;
```
