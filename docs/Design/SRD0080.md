## Rule Information
  
|    |    |
|----|----|
| Id | SRD0080 |
| Friendly Name | TOP expression should use parentheses |
| Category | Design |

| Applicable Types | Procedure  |
|   | Scalar Function |
|   | Table Valued Function |

## Description
  
Expression used with TOP should be wrapped in parentheses.
  
## Summary
  
Expressions used with TOP should be wrapped in parentheses.
  
### Examples
  
```sql
CREATE PROCEDURE [dbo].[SelectTopNoParen]
	
AS
	set nocount on;
	Select top 100 Col1 from dbo.TestTableSSDT;
RETURN 0

-- SRD0080, SRD0014
```
```sql
CREATE PROCEDURE [dbo].[TopExpressionParenthesesTest]
AS
BEGIN
    SET NOCOUNT ON;
    SELECT TOP 100 Col1
    FROM [dbo].[TestTableSSDT]
    ORDER BY Col1;
END;
RETURN 0;

-- SRD0080
```
  
### Remarks
  
Wrapping TOP expressions in parentheses avoids syntax issues and makes complex TOP clauses
clearer and more consistent.
