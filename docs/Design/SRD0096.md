## Description
  
Potential SQL injection issue.
  
## Summary
  
Dynamic SQL built from procedure input can introduce SQL injection vulnerabilities.
  
### Examples
  
```sql
CREATE PROCEDURE [dbo].[Injection]
	@param1 varchar(255)
AS
SET NOCOUNT ON;
	exec('Select * from table where name like '''+@param1+'''');
RETURN 0

	@param1 varchar(255)

SET NOCOUNT ON;

    exec(@sql);
RETURN 0
```
