## Description
  
Avoid using SELECT *.
  
## Summary
  
Avoid the use of SELECT *. This can cause more columns than desired to be returned, and can also cause issues if the order of the columns changes.
  
### Examples
  
```sql
Create view [dbo].[err]
	AS SELECT * FROM [TestTableSSDT]
```
